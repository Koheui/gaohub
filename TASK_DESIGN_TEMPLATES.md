# 🎨 Neo EventHub: LP & Banner デザインテンプレート多角化・パターン拡充タスク (Claude Code 開発指示書)

## 📌 概要
本タスクは、Neo EventHub（GAO HUB）におけるイベント公開LPおよびSNS告知用バナーのデザインが単一のトーン（Kodak等）でワンパターンに見えてしまう課題を解消し、多様なイベントブランド（テックカンファレンス、ラグジュアリーサミット、音楽フェス、ビジネスセミナー等）に対応できる**先進的テンプレートバリエーションの量産**を目的とします。

Claude Code を用いて、以下のタスクを順次実装・拡充してください。

---

## 🛠️ 開発タスク一覧 (Action Checklist)

### 1. LPテーマテンプレートの追加・多角化 (`src/components/lp/theme.ts` & `src/app/e/[slug]/page.tsx`)
既存の3テーマ (`kodak`, `spectrum`, `aurora`) に加え、以下の4テーマを新規実装し選択可能にする。

- [ ] **`neon-cyber` (サイバー・ネオンダーク)**
  - 漆黒背景 (`#09090b`) に鮮やかなシアン/ピンクのネオン発光グラデーションとグリッドライン。
  - サイバー感溢れる未来志向のテックカンファレンス向け。
- [ ] **`swiss-minimal` (スイス・タイポグラフィ)**
  - 純白/ライトグレー背景に超太字ゴシック体とビビッドな単色アクセント（レッド/イエロー）。
  - クリーンで余白を極めたインターナショナル・タイポグラフィ・スタイル。
- [ ] **`editorial-serif` (エディトリアル・ラグジュアリー)**
  - ディープグリーン/ネイビー地にセリフ体（明朝体フォント）を掛け合わせた高級感あるサミット向け。
- [ ] **`metro-grid` (ブルータリズム・グリッド)**
  - 太いブラックボーダーとブロック分割、強烈なコントラストのポップアート調レイアウト。

---

### 2. 告知バナー生成テンプレートの追加 (`src/lib/server/bannerImage.tsx`)
Satori (`@vercel/og`) で動的生成されるバナースタイル (`classic`, `workandrole`, `duotone`, `geo`) に、以下のデザインパターンを追加入力可能にする。

- [ ] **`type-heavy` (タイポグラフィ・ポスター)**
  - 登壇者写真を背景に大きく透かし、画面いっぱいにセッションタイトルを巨大文字で配置するポスター調デザイン。
- [ ] **`monochrome-minimal` (モノクロ・エディトリアル)**
  - 背景を無地の白/黒に切り詰め、白黒写真と洗練された直線グリッドのみで構成されるミニマルデザイン。
- [ ] **`split-duotone` (左右2色分割デュオトーン)**
  - 左側と右側で異なるテーマカラー（例: オレンジ×ネイビー）を大胆に分割配置し、人物写真を二色グラデーションで染め上げるデザイン。

---

### 3. ダッシュボード管理画面でのプレビュー・切り替え UI 拡張
- [ ] **`src/components/EventForm.tsx`**
  - テンプレート選択に新規テーマ (`neon-cyber`, `swiss-minimal`, `editorial-serif`, `metro-grid`) を表示し、スウォッチ・プレビュー付きで選択可能にする。
  - 統計/カウントダウンの非表示（`statsStyle: "none"`）切り替えを分かりやすく案内。
- [ ] **`src/app/dashboard/events/[id]/banner/page.tsx`**
  - バナースタイル切り替えメニューに新スタイルを追加し、リアルタイム生成プレビューを可能にする。

---

### 4. 【将来拡張タスク】EC物販ショップ・テンプレートの追加 (`src/components/shop/`)
主催者が自社商品（小倉コーラ原液、emolink物理カード、イベント限定Tシャツ、自著等）を販売する際のショップページテンプレート。

- [ ] **`shop-editorial` (エディトリアル・ストーリー型ショップ)**
  - 大判写真とプロセスエコノミー文脈（ストーリー）を中心に伝える高級感あるクラフトショップ。
- [ ] **`shop-street` (ストリート・グラフィック型ショップ)**
  - 太枠・ビビッドカラー・ダイナミックなカードレイアウト。イベントグッズ・アパレル向け。
- [ ] **`shop-minimal` (ミニマル・グリッド型ショップ)**
  - 余白重視のクリーンな3列グリッド。デジタルコンテンツ（有料動画・資料DL・電子チケット）向け。
